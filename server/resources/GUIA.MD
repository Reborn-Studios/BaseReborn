## Mudanças

### Alterar Farms:

`[ Reborn ] > ld_routes > config.lua`

`[ Gerais ] > Farms > Config.lua`

`/addgarage` - Comando para criar garagem dentro do jogo

`[ Reborn ] > will_garages_v2 > Garages.lua / Vehicles.lua`

`[ Scripts ] > ox_inventory > data > items.lua`
